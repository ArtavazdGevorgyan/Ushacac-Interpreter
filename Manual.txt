#      Programming Language

       Ushacac Interpreter

Ushacac Interpreter լեզվով գրված ծրագիրը ունի ․ush ընդլայնումը

1)Փոփոխական հայտարարաելուց օգտագործվում է vrbl հրամանը․
    Օրինակ՝
        vrbl a կամ
        vrbl a = "Hello" արդն վերագրված արժեքով կամ
        vrbl a = 55 արդն վերագրված արժեքով
2) Ushacac Interpreter-ում կան բարդ արտահայտություններ․
    Օրինակ՝
        a = ( 2 + 2 ) * 3 կամ
        vrbl b = 21 + 9 / ( 5 + 25 ) 
3) էկրանին ինֆորմացիան ցույց տալու համար օգտագործվում է output հրամանը․
    Օրինակ՝
        output a կամ
        output "Hello" կամ
        output ( 2 + 2 ) * 3 կամ
4) ՈՒշադրություն դարձրեք բացատներին քանի-որ առանց նանց ծրագիրը չի աշխատր կամ կաշխատի սխալ․
    Օրինակ՝
        output (2+2)*3      այս դեպքում ծրագիրը չի աշխատր
5) Այս լեզվում կան նաև պայմանի օպերատորներ․
    Գրելաձև՝
        if condition 
        {
            instructions
        }
        else
        {
            instructions
        }
    Որտեղ condition-ը պետք է առանձնացված լինի բացատներով
    Օրինակ՝ a > b
6) Այս լեզվում կա նաև ցիկլ․
    Գրելաձև՝
        loop condition
        {
            instructions
        }


Ծրագիրը աշխատացնելու համար պետք է բացել Terminal-ը և գրել՝

        python main.py code.ush
